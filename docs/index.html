🔍 내 맥북 종류 확인하는 법
M1/M2/M3 화면 왼쪽 위 애플 메뉴() → 이 Mac에 관하여 → 칩 항목에 Apple M 표시
Intel 화면 왼쪽 위 애플 메뉴() → 이 Mac에 관하여 → 프로세서 항목에 Intel 표시
⚙️ 설치 방법
1다운로드된 .zip 파일 더블클릭 → 앱 파일 추출
2앱 아이콘을 우클릭 → 열기 클릭 (처음 한 번만)
3"개발자를 확인할 수 없습니다" 경고창에서 열기 클릭
4아이디와 비밀번호로 로그인
