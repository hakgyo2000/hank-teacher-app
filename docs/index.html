🔍 내 맥북 종류 확인하는 법
M1/M2/M3 애플 메뉴() → 이 Mac에 관하여 → 칩 항목에 Apple M 표시
Intel 애플 메뉴() → 이 Mac에 관하여 → 프로세서 항목에 Intel 표시
⚙️ 설치 방법
1다운로드된 .zip 파일 더블클릭 → 두 개 파일 압축 풀림
2「Hank DP 설치완료.command」 파일 우클릭 → 열기 클릭
3경고창에서 열기 클릭 → 터미널이 자동으로 설치 완료 처리
4앱이 자동으로 실행됩니다. 이후부터는 앱 아이콘 더블클릭으로 실행
5아이디와 비밀번호로 로그인
